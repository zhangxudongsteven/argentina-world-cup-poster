# Argentina World Cup Poster

一张致敬阿根廷国家足球队三次世界杯夺冠历程的响应式网页海报。

## 本地预览

直接打开 `index.html`，或在项目目录运行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 部署到 GitHub Pages

1. 将此目录提交并推送到 GitHub 仓库。
2. 进入仓库的 **Settings → Pages**。
3. 在 **Build and deployment** 中选择 **Deploy from a branch**。
4. 选择 `main` 分支和 `/ (root)` 目录并保存。

页面不依赖构建工具，GitHub Pages 可直接发布。
